Hiking Planner

📸 Scenic

Data sources: Google Places API (location, name, rating, photo URLs) for highly-rated POIs ≥4.5 stars with ≥20 reviews; OpenStreetMap (manual hiking/scenic landmarks, viewpoints, natural features).

🥾 Trails

Data source: OpenStreetMap (hiking routes, trekking paths with difficulty=sac_scale ratings and surfaces).

🗺️ Terrain

Data sources: USGS elevation data and Google Maps (elevation in meters, used to generate topographic contours and hillshading).

⛰️ Steep

Data source: Slope analysis (>15° inclination) calculated from USGS elevation data for alpine/technical route awareness.

🚐 Parks

Data source: OpenStreetMap (camping_site and caravan_site amenities with facilities and accessibility info).

🪙 Tolls

Data source: OpenStreetMap (road segments tagged toll=yes with toll operator names and fee info where available).

💧 Water

Data source: Overpass API querying OpenStreetMap (amenity=drinking_water and water_point locations with wheelchair access info).

🎟️ Activities

Data source: Amadeus Activities API (activity discovery and pricing in current map area).

🍄 Foraging

Data sources: Custom edible mushroom database (32 species profiles); Open-Meteo (7-day weather: temperature, rainfall); SoilGrids (soil pH); EPFD/Earth Engine CORINE (forest type and vegetation class). Probability score (0-100%) filters species by temperature tolerance, rainfall, soil pH, tree habitat, and land cover type. Only non-toxic species shown.

⛽ Gas

Data source: HERE Routing API (fuel station POIs with brand names and fuel types; available in selected European countries only).

🔧 Car Repair

Data source: Google Places API (text search for "car repair" POIs with rating ≥ 4.2). Optional filters can restrict results to shops with at least N scenic spots within R km (only available when the Scenic layer is on).
